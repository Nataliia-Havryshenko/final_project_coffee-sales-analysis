# final_project
## Опис проєкту
### Аналіз продажів кави з метою:
- Виявити пікові години продажів
- Визначити найпопулярніші напої та категорії
- РОзглянути промо активності для стимулювання росту  продажів
## Дані
- Файл: `data/coffee_sales.csv`
- Колонки:
  - `date`, `datetime`, `cash_type`, `card`, `money`, `coffee_name`
  - Додаткові: `weekday`, `month`, `is_weekend`, `category`, `hour_of_day`, `peak_period`, `time`, `day_type`, `loyalty_customer`

## Використані інструменти
- Python (pandas, matplotlib, seaborn)
- Power BI (для інтерактивного дашборду)

## Структура репозиторію
- `data/` – сирі дані
- `notebooks/` – jupyter notebooks
- `scripts/` – скрипти Python
- `reports/` – графіки та результати

## Візуалізації
- Продажі по годинах дня
- Продажі по пікових періодах
- Топ-напої
- Середній чек по категоріях
- Продажі по днях тижня
- Продажі по типу платежу
- Продажі по пікових періодах та категоріях

## Висновки та рекомендації

1. Morning/Lunch/Evening Peak — пікові години продажів, варто фокусувати промо у ці періоди.
2. Середній чек найвищий ввечері → можливість просувати дорогі напої.
3. Вихідні мають менше продажів → розглянути комбо-пропозиції.
4. Переважна більшість оплат карткою → зосередитися на безготівкових платежах.
5. Розглянути графік працівників в години пік додатковий вихід, для швидкого обслуговування покупців.
